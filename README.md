# Serenova AI 🌿

Serenova AI is an intelligent mental wellness assistant designed to help users understand their emotional state and receive supportive, calming guidance. The app simulates AI-driven emotional analysis using user input and provides a calm, beautiful, and interactive user experience.

## Features ✨
- **Mood Detection Logic**: Analyzes user input to detect emotions such as stress, sadness, anxiety, and happiness using keyword detection.
- **Mood Scoring**: Generates a dynamic Mood Score based on the detected emotion.
- **Supportive Insights**: Displays positive, supportive AI-generated messages tailored to the user's emotional state.
- **Wellness Suggestions**: Offers actionable mental wellness tips like deep breathing, short walks, or listening to music.
- **Mood History Tracker**: Beautifully logs recent queries and sentiments in a tracking list.
- **Calm Mode**: A focused, immersive full-screen breathing exercise overlay (Inhale... Exhale...).
- **Modern UI/UX**: Soft gradients, glassmorphism cards, interactive micro-animations, color-coded sentiment analysis, and fully responsive design.

## Tech Stack 🛠️
- **HTML5**: Semantic layout and structure.
- **CSS3**: Variables, Flexbox/Grid, animations, glassmorphism styling (`backdrop-filter`).
- **JavaScript (ES6)**: DOM manipulation, timing events for breathing, keyword algorithms.

## How to Run 🚀
1. Clone the repository or download the files.
2. Open the `SerenovaAI` directory.
3. Simply double-click on `index.html` to open it in any modern web browser.
4. (Optional) Run using a local development server like VS Code Live Server or `npx serve`.

## Future Improvements 🔮
- **Real AI Integration**: Hook up to an LLM API (like Gemini or OpenAI) for more nuanced and dynamic responses instead of keyword matching.
- **Persistent Storage**: Use `localStorage` or a database to save mood history across sessions.
- **Charts & Analytics**: Visualize mood trends over time using a library like Chart.js.
- **Audio Integration**: Add soothing ambient sounds that play during the "Calm Mode" breathing exercise.
